### 二、 使用

#### 2.1 圆形图片属性

~~~
  qiyou:q_img_scr 设置要显示的图片  
  qiyou:q_circle_border_width 设置外边框宽度 默认5dp    
  qiyou:q_ciecle_color 设置外边框颜色 默认黑色    
  qiyou:q_show_border 设置是否显示外边框 true显示 false不显示 默认为true  
~~~
